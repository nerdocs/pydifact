# Zips directory

This is the directory where pydifact-generator searches for sources.

They are not included within pydifact, as they are mostly available at the UNECE and GEFEG websites.

Download all necessary sources from:

* https://unece.org/trade/uncefact/unedifact/download
* https://service.gefeg.com/jwg1/Current/id1.htm:
  * https://service.gefeg.com/jwg1/Archive/v3/data/v3.html
  * https://service.gefeg.com/jwg1/Archive/v4/data/v4.html
* https://service.gefeg.com/jwg1/Archive/cl/data/unsl{directory}.zip


The generator will try to find all necessary zips for the given release and will complain if any needed is missing.
